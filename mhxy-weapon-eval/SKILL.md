# 梦幻西游武器价值评估

---

## 一、输入方式

用户通常以**截图**方式提供武器信息，AI 需要进行 OCR 识别，从图中提取以下属性字段：

### 必须提取的字段

| 字段 | 说明 | 示例 |
|-----|------|------|
| 等级 | 装备要求等级 | 150 |
| 五行 | 金/木/水/火/土 | 金 |
| 价格 | 用户提供的售价/标价（梦幻币） | 5,000,000 |
| 命中 | 武器命中值 | 312 |
| 伤害 | 武器伤害值 | 789 |
| 耐久度 | 当前耐久/最大耐久 | 180/200 |
| 开孔孔数 | 已开的符石孔数量 | 3 |
| 符石组合 | 已镶嵌的符石及等级 | 3级红玛瑙符石 |
| 锻炼等级 | 宝石锻炼等级 | 5 |
| 镶嵌宝石 | 宝石类型（由锻炼等级推断等级） | 太阳石 |
| 属性加点 | 正数或负数均可能出现 | 魔力+15, 力量+10, 敏捷+20, 耐力-11, 体质-20 |
| 特技 | 装备附带的特技名称 | 四海升平 |
| 修理失败次数 | 0~3次 | 1 |

### OCR 提取注意事项
- 属性加点可能为**正数也可能为负数**，务必保留正负号
- 五维属性（魔力、力量、敏捷、耐力、体质）不一定全有，缺少的视为 0
- 特技可能没有（普通武器）
- 修理失败次数如果截图中未显示，默认为 0
- 价格如果截图中没有，需**主动询问用户**

## 二、评估计算（共九项）

> 默认使用 **2025区** 的宝石价格数据（来自 mhxy-gem-price skill）。

### 第一项：综合伤害

```
综合伤害 = 命中 / 3 + 伤害
```

### 第二项：综合物伤

物理伤害受**种族**影响，力量转化系数不同：

| 种族 | 力量系数 | 公式 |
|-----|---------|------|
| 人族 | 0.67 | 力量 × 0.67 + 综合伤害 |
| 魔族 | 0.77 | 力量 × 0.77 + 综合伤害 |
| 仙族 | 0.57 | 力量 × 0.57 + 综合伤害 |

```
综合物伤 = 力量 × 种族系数 + 综合伤害
```

> 注意：力量值从属性加点中读取，可能为负数。如果没有力量加点则力量=0。
> 输出时**三个种族都算出来**，方便用户参考。

### 第三项：综合法伤

```
综合法伤 = 魔力 × 0.7 + 力量 × 0.4 + 体质 × 0.3 + 耐力 × 0.2 + 综合伤害 / 4
```

> 注意：各属性从属性加点中读取，可能为负数，缺少的属性视为 0。

### 第四项：武器裸价

```
武器裸价 = 售价 - 已投入成本
```

> - **武器裸价**：武器本身的价值，不含宝石、开孔、符石等附加投入
> - **已投入成本** = 宝石成本 + 开孔成本 + 符石成本
> - 如果武器无宝石无孔，则武器裸价 = 售价

### 第五项：单点伤害售价

```
单点伤害售价 = (售价 - 宝石成本) / 综合伤害
```

> - 数值可能为**负数**（宝石成本高于售价时）
> - **数值越小越好**，负数越大（绝对值越大的负数）越好，说明性价比越高
> - 如果用户未提供价格，此项无法计算，需提示用户补充

### 第六项：每百万伤害

```
每百万伤害 = 综合伤害 / (武器裸价 / 100万)
```

> - **每百万伤害**：每花100万能买到多少综合伤害
> - **数值越高越好**，说明性价比越高
> - 用于对比不同武器的伤害性价比

### 第七项：宝石成本

根据武器上的**锻炼等级**和**镶嵌宝石类型**，查询 mhxy-gem-price skill 中对应服务器的价格计算。

计算规则：
- 锻炼等级 = 已镶嵌的宝石等级总和（每颗+1级）
- 宝石类型从"镶嵌宝石"字段读取
- 查 mhxy-gem-price 中该宝石的1级基准价 P
- 宝石成本 = P × 2^(锻炼等级-1)（宝石公式）

> 如果武器镶嵌了两种宝石，需分别计算后相加。

### 第八项：开孔成本

不同等级装备的开孔规则：

| 装备等级 | 最多开孔数 | 每次费用 |
|---------|----------|---------|
| 0-30级 | 1孔 | 10万 |
| 40-60级 | 2孔 | 20万 |
| 70-90级 | 3孔 | 30万 |
| 100-120级 | 4孔 | 40万 |
| 130-150级 | 5孔 | 50万 |

计算两个成本：
```
最低开孔成本 = 开孔数 × 对应等级段的每次费用
平均开孔成本 = 最低开孔成本 × 3（考虑失败概率）
```

### 第九项：修理失败次数

**必须醒目展示**，直接影响装备价值。

| 失败次数 | 影响说明 |
|---------|---------|
| 0次 | 正常装备，无影响 |
| 1次 | 轻微影响，注意后续修理风险 |
| 2次 | 明显减值，修理需谨慎 |
| 3次（上限） | 无法通过普通修理恢复耐久，仅可通过珍珠/乾坤石/快捷修理恢复。**大幅减值** |

> 修理失败不会影响装备属性，但影响耐久恢复方式和装备交易价值。

### 第十项：特技

直接展示特技名称。如果有特技，是武器的重要增值项。无特技则标注"无"。

### 第十一项：属性加点

列出所有属性及数值（含正负），格式如：

```
魔力: +15
力量: +10
敏捷: +20
耐力: -11
体质: -20
```

## 四、计算示例

假设用户提供一把武器：
- 等级：150，伤害：789，命中：312
- 力量+10，魔力+15，体质-20，耐力-11，敏捷+20
- 锻炼等级5，镶嵌太阳石，开孔3孔
- 售价：5,000,000，修理失败0次，无特技

**计算过程：**

1. **综合伤害** = 312/3 + 789 = 104 + 789 = **893**
2. **综合物伤**：
   - 人族：10 × 0.67 + 893 = 6.7 + 893 = **899.7**
   - 魔族：10 × 0.77 + 893 = 7.7 + 893 = **900.7**
   - 仙族：10 × 0.57 + 893 = 5.7 + 893 = **898.7**
3. **综合法伤** = 15×0.7 + 10×0.4 + (-20)×0.3 + (-11)×0.2 + 893/4 = 10.5 + 4 - 6 - 2.2 + 223.25 = **229.55**
4. **宝石成本**（2025区太阳石，5级）= 122004 × 16 = **1,952,064**
5. **已投入成本** = 1,952,064 + 1,500,000 + 0 = **3,452,064**
6. **武器裸价** = 5,000,000 - 3,452,064 = **1,547,936**
7. **单点伤害售价** = (5,000,000 - 1,952,064) / 893 = **3,415.5 梦幻币/点**
8. **每百万伤害** = 893 / (1,547,936 / 1,000,000) = **576.8**
9. **开孔成本**（150级装备3孔）：最低 = 3 × 50万 = **150万**，平均 = **450万**

## 五、特殊情况处理

1. **用户未提供价格**：跳过"单点伤害售价"计算，提示用户补充后可继续计算
2. **无锻炼等级/无宝石**：宝石成本为 0
3. **无开孔**：开孔成本为 0
4. **属性全为负数**：正常计算，综合物伤/法伤可能为负值，如实展示
5. **截图模糊无法识别**：列出无法识别的字段，请用户手动补充
6. **非2025区用户**：提示用户指定服务器区，如该区无宝石价格数据则无法计算宝石成本
