# 梦幻西游游戏介绍

> 以下内容基于《梦幻西游》电脑版官网资料整理

## 一、背景故事

话说数千年前，蚩尤复活动乱，挑拨仙魔大战，人类为求自保，或投身于两方之一，或自成一伍，一场人、仙、魔三界混战就此拉开帷幕，三界间战火屠天，秩序不复存在。战争终以蚩尤阴谋被揭发为终，人仙魔三族英雄携手将蚩尤打回战神山武神坛下，封印千年。

时至大唐贞观年间。三界大战已平息五百余载，世间重又恢复了安宁与繁荣。当世分为**四大部洲**，众生善恶不一：

- **东胜神洲**：居生奇兽灵仙，汲日月精华，敬天地礼法，心爽气平
- **北俱芦洲**：为蛮族所踞，多生凶禽猛兽，异常凶险
- **西牛贺洲**：多隐仙庭道馆，习阴阳之术，养气潜灵人人固寿
- **南瞻部洲**：独为人世红尘之所，亦为人、魔、仙三族混居之地

天地间芸芸众生分为**人、仙、魔**三大种族，各族天赋不同，自有所长：

- **仙族**：超脱而飘逸，能够掌握天地之灵气，并能将他们转化成为百变神通
- **魔族**：从阴暗中走出，崇武好杀，往往拥有超强的战斗力和抵抗力
- **人类**：看似平凡，在战火中习得阴阳五行之理，演化为玄妙的法术和武学，并尝试挖掘自己强大的心灵力量，立身于三界之间

三族均各自创立四大门派，以广纳门徒，传授技艺，壮大声威。

**观音大士**观众生相，三界大战之后，心魔肆虐，尘世中人性的莲花已经枯萎。邪恶与欲望逐渐滋长，吞噬了心灵间的良善，将巨大的潜力引入无边的黑暗，而佛主如来处有三藏真经，可以劝人为善，待要送上东土，只怕众生愚昧，毁谤真言，故决定广募与佛有缘之人，遍大义于天下，寻找**五个天命取经人**。

《梦幻西游》的故事由此展开，玩家将在经历一段际遇后得到金莲花，成为有佛缘之人，接受寻找天命取经人，帮助他们拂开尘眼，见性归真的使命。

这五个天命取经人分别是：

1. **金蝉法师**：佛祖座下被贬下凡尘
2. **天蓬元帅**：调戏嫦娥犯了天条，被打入人间投了猪胎
3. **卷帘大将**：失手打碎琉璃盏惹怒王母，被罚下人间受苦
4. **小白龙**：受冤背屈的神龙之子
5. **孙悟空**转世：五百年前大闹天宫，令各路神仙闻风丧胆的妖猴

这五人或有凄凉身世，或有无限心结，更或全忘前世因果，变得愚钝不可教化。四海之大，该向何方寻找，该如何劝服他们甘愿苦历千山，远经万水，到佛祖处求取真经，劝化众生，化解千年冤戾。一切都是未知，都等待着玩家的探寻。

## 三、游戏特色

### 门派系统

《梦幻西游》的世界中仙、人、魔三族各有传承。为了让自己的门派弟子横行天下，各派师傅都不吝一己之私，纷纷将压箱底的功夫拿出来教育门徒。同时《梦幻西游》中首次鲜明地引入了**门派克制系统**，三族十五派相互克制，每个门派都有一招必杀技来克制另外一个门派，环环相扣，成为了一个结构严谨的"门派克制链"，从根本上杜绝了一派过于壮大的可能性。

### 帮派系统

四海之大，任你鱼跃。在《梦幻西游》世界里，你可以孤身游历江湖，作一名无牵无挂的独行侠客，也可以携手四方朋友，组建一支属于自己的帮派，共同体验闯荡江湖带来的乐趣。帮派是体现游戏互动性的一项重要内容，在这里你可以体会到团体带来的荣誉感和归属感。

### 结拜系统

千金易得，挚友难求。武林儿女闯荡江湖，能够结交到知心的好友是最珍贵的收获！结拜好友除了在共同战斗时提高自动保护、合击几率外，还能获得独特的自制称谓！

### 婚姻系统

千里姻缘一线牵，在《梦幻西游》的世界里，最浪漫而难忘的一刻，就是和自己的心上人一起携手，步入婚礼的殿堂，结为人人羡慕的伴侣。一对新人穿上喜庆的大红新装，在鸳鸯祠前拜堂成亲，月老在婚礼现场洒下喜糖给四方宾客，整个婚礼场面可谓热闹非凡。

### 欢乐家园

这里是又一片广阔的游戏空间：幽雅舒适的新居，趣味盎然的游戏氛围，尽情体验动手建房子、做家具的快乐，玩家建造的房屋不仅拥有各项丰富实用的功能，还能与您的心上人享受共同布置温馨小家的乐趣！

### 养育系统

无论是已经结婚的玩家，还是单身玩家，都能够获得自己的宝宝，体验生儿育女的天伦之乐！宝宝将来会像父亲一样勇敢威风，还是像母亲一样美丽温柔，都取决于玩家的养育方式。

### 商业系统

游戏中有着各种店铺、酒楼、钱庄、当铺，为玩家提供交易、存款及娱乐等服务，还有各种商人出现在各大城市之中。此外，玩家还可以自己摆摊，或者开设属于自己的商店，体验经营店铺的辛苦与乐趣。

### 任务系统

无论你是初涉江湖的新人，还是修道有成的高手，总能找到适合自己的任务。除了一些随时可做的常规任务外，游戏中还有一些定期类的活动，趣味十足奖励丰厚，不容错过。

### 坐骑系统

各位英雄侠少闯荡在梦幻西游中，坐骑天下带给您丰富多样的坐骑选择——或威风凛凛或玲珑可爱的坐骑伴随身边，从此江湖路上不再寂寞。

## 四、画面风格

### 人物造型

游戏的人物造型为**Q版造型**，非常可爱。游戏里有三个种族共**十五个人物角色**可以选择，分别是八男七女：

- **男性人物**：英武神勇的神天兵、潇洒飘逸的剑侠客、憨厚可爱的虎头怪等
- **女性人物**：娇小可爱的骨精灵、灵气逼人的玄彩娥、不让须眉的英女侠等

代表角色包括：逍遥生、骨精灵、龙太子、神天兵、虎头怪等。

### 场景画面

网易的美工成功地将中国古典建筑的凝厚与Q版的意趣风格糅合在一起，将《梦幻西游》的场景打造成一幅真正意义上的"游戏画卷"。游戏中所有人物的3D造型都显得异常的圆润。同时游戏场景的构成手法也非常老到，无论是干涩少雨的沙漠，还是湿润凝重的绿洲，在游戏中都显得栩栩如生。

### 版图规模

《梦幻西游》中的版图非常庞大，一共有**四个大洲、五组迷宫及十八个门派**以供玩家来探险，每一个地方都有独特的特色：

- 一望无垠的边境沙漠
- 盛世大唐的长安城
- 终年冰雪覆盖的北俱芦洲
- 阴森诡异的阴曹地府
- 波涛汹涌的东海龙宫

各个地方的建筑特色也不尽相同，玩家可以感受到不同的文化韵味。游戏画面极为精细，加上整幅构造的地图，有着很强的整体感。游戏中所有场景都由美工手绘完成。

## 五、新手入门

### 5.1 登录游戏

1. **第一步**：运行《梦幻西游Online》游戏程序，点击"登录"
2. **第二步**：阅读游戏公告，点击"下一步"
3. **第三步**：通过"扫码登录"或"普通登录"登录游戏账号
   - **扫码登录**：使用手机打开"网易将军令"或"网易大神"，扫描客户端二维码，选择要登录的账号，核对信息无误后，点击"确认登录"
   - **普通登录**：输入网易通行证账号、密码，确认无误后点击"下一步"
4. **第四步**：点击要登录的服务器，即可直接进入该服务器。若点击"下一步"，则进入默认服务器
5. **第五步**：若在该服务器已有角色，在右侧选择要登录的角色后，点击"进入游戏"即可开始游玩。若是初次进入该服务器，需要阅读"服务条款、玩家守则、隐私政策"各项条款后，点击"我接受"
6. **第六步**：选择自己喜欢的角色与门派，输入角色昵称，确认无误后，点击"下一步"即可开启梦幻新旅程

**特别说明**：为了保护玩家账号信息的安全、提高登录的便捷性，普通登录界面增加了"密码保护"和"记住账号"功能：

- **密码保护**：启用后，如果有其它程序正在监视玩家的键盘输入，该功能将自动启用保护程序，令玩家无法在登录界面中输入账号和密码，提醒玩家确认相应程序的合法性，达到防范盗号的目的
- **记住账号**：勾选后，下次登录无需再次输入账号。公共环境下请谨慎使用

### 5.2 基本操作

| 操作 | 方法 |
|------|------|
| **人物行走** | 将鼠标停在想要走到的地方，点一下鼠标左键即可；也可以按下鼠标左键停留两秒钟，角色就会跟随鼠标指向移动 |
| **人物转向** | 角色站立不动时，点击鼠标右键可以命令角色转向鼠标所指的方向 |
| **拾取物品** | 地面有掉落的物品时，走到物品旁边，鼠标左键点击物品即可拾取 |
| **使用物品** | 左键点击画面下方的小箱子（快捷键 `Alt+E`），打开道具栏，将鼠标移到某件物品上，再点一下右键即可使用 |
| **摧毁物品** | 在道具栏中左键点选某件物品，抓起后移到道具栏以外，再点一下左键可以将物品摧毁（需要确认一次） |
| **与NPC对话** | 走到NPC旁边，将鼠标停留在NPC身上，点一下左键即可与其对话 |
| **屏蔽玩家** | 点击 `F9` 键可以屏蔽其他玩家和他们的发言，更方便的点击NPC |
| **屏蔽摊位** | 使用快捷键 `Alt+H` 可以屏蔽玩家摊位 |

### 5.5 常见问题

**Q：游戏中如何升级？**

人物角色通过积累经验来升级，在当前经验满足升级所需的时候可以选择等级提升，升级后将获得10点属性点数，其中5点平均分配到体质、魔力、力量、耐力和敏捷上（每项1点），其余5点由玩家根据自己的意愿自行分配。召唤兽在经验达到要求时等级自动提升，升级后获得10点属性点数，分配方法和人物角色相同。宠物的经验根据在线时间计算，经验达到要求后自动升级。

**Q：如何获得经验？**

经验可以通过打怪积累，也可以做一些任务得到，例如游戏初期的新手任务、师门任务等，另外参加游戏中的某些活动也可能得到经验奖励。

**Q：宠物有什么作用，怎样得到宠物？**

宠物可以携带和驯养召唤兽，进行召唤兽合成。在玩家第一次进入游戏的地方有一位宠物仙子，和她交谈后便可选择一只喜欢的宠物来领养。

**Q：什么是召唤兽，如何获得召唤兽？**

召唤兽可以被召唤出来协助战斗，玩家需要拥有宠物才可以携带召唤兽。游戏中出现的所有怪物，除了在任务中出现的特定怪物外，大部分都可以被玩家捕获而成为召唤兽，也可以通过交易向其他玩家购买。

**Q：如何获得装备？**

部分装备可以在店铺中购买，平时战斗、完成任务和参加活动过程中也有可能得到装备奖励，游戏后期玩家可以通过自行打造得到性能更好的装备。

**Q：如何获得药品？**

游戏中的药品按照作用效果的强弱分为三个等级：

- **一级药品**：可从店铺直接购买，恢复效果较弱
- **二级药品**：可以在战斗中获得，恢复效果比较强
- **三级药品**：通过炼药得到，恢复效果强于二级药品，有些三级药品还具有解除异常状态、复活等作用

**Q：角色携带金钱上限是多少？**

每个角色携带的现金+存款不能超过 **（等级×等级×2000+10000）**，最高上限为 **2000万两**，其中存款上限为 **1000万两**，超过这个数值的存款无法存入钱庄。当携带金钱超过上限时，会得到系统提示，多出来的部分在下线5分钟后登陆时将会转化为储备金。

**Q：食物和药品有什么作用？**

部分药品和食物只能在战斗中或平时状态下使用，其它药品食物在两种情况下均可使用。主要用来恢复人物或召唤兽的状态，部分药品还具有复活、解毒等特殊效果。

**Q：怎样进行游戏截图？**

游戏时按下 `PrintScreen` 键可以保存当前游戏画面，保存的图片放在游戏目录下的 `screen` 文件夹中。

**Q：怎样鉴定装备？**

未鉴定的装备无法查看到属性，暂时也不能使用，需要进行鉴定后才可佩带：

- **武器和铠甲**：可以使用大唐官府的门派法术"兵器谱"进行鉴定
- **防具类物品**（包括头盔、靴、腰带、饰品）：可以使用地府的"勘察令"法术进行鉴定
- **初级装备**：可以找游戏中的鉴定商鉴定

**Q：体力和活力有什么作用？**

- **体力**：打造、打工时会消耗体力，初始值为10，上限受到玩家等级和辅助技能"健身术"等级的影响，玩家在线情况下体力会逐渐恢复
- **活力**：炼药、烹饪、鉴定物品时需要消耗一定的活力值，初始值为10，上限受到玩家等级和辅助技能"养生之道"等级的影响，玩家在线情况下活力会逐渐恢复

> 提示：在游戏初期可以使用"烹饪"消耗活力获得道具"包子"。在非战斗状态下使用包子可以恢复100点气血。

**Q：如何变换服装的颜色？**

第一次创建游戏角色时，可以根据自己的爱好为角色选择外观色彩。另外在游戏中也可以找长安城的配色师更换服饰的颜色，更换时需要有道具"花豆"、"彩果"作为颜料。

### 5.6 点数分配

游戏中人物的天赋属性有五个，包括**体质、魔力、力量、耐力和敏捷**，天赋属性的变化影响着角色在各方面的能力强弱。人物属性的初始值由系统赋予，不同种族的角色，初始的属性也不同。

角色每次升级时，可以获得 **10点潜力**，其中5点平均分配到五种属性上，剩余5点由玩家根据意愿自行分配。

#### 五大属性说明

| 属性 | 效果 |
|------|------|
| **体质** | 对生命值的多少有很大影响，增加体质点数可以提高气血上限，并增加些许灵力 |
| **魔力** | 法力高低的代表，增加魔力点数可以增加魔法上限，并提高灵力 |
| **力量** | 增加力量点数可以提高命中和伤害，并增加些许灵力 |
| **耐力** | 对物理攻击的抵抗能力，增加耐力点数可以提高物理防御，并略微增加灵力 |
| **敏捷** | 增加敏捷点数可以提高躲避和速度，从而更容易掌握战斗的先机 |

> 属性影响程度：红色连线 = 影响大，蓝色连线 = 影响中，绿色连线 = 影响小

#### 种族天赋属性发挥

| 种族 | 体质发挥 | 魔力发挥 | 力量发挥 | 耐力发挥 | 敏捷发挥 |
|------|----------|----------|----------|----------|----------|
| 人族 | 均衡 | 均衡 | 均衡 | 均衡 | 均衡 |
| 仙族 | 弱 | 强 | 较弱 | 较强 | 均衡 |
| 魔族 | 强 | 弱 | 较强 | 较弱 | 均衡 |

#### 种族先天赋予点数

### 5.7 游戏时辰

在中国传统文化中，一昼夜划分为十二个时辰。十二个时辰用十二地支的名字命名，即子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。《梦幻西游》的世界也是由十二个时辰构成的。

#### 时辰显示

游戏屏幕的左上角显示时辰信息：左边的图案显示当前时辰，右边的图案区分白天与黑夜。

#### 时辰规则

1. 游戏中时辰是以服务器时间为基准计算的
2. 从整点开始刷新时辰：子时→丑时→寅时→卯时→辰时→巳时→午时→未时→申时→酉时→戌时→亥时，如此重复交替
3. 游戏中每个时辰相当于现实时间 **2分30秒**
4. **白天**时辰：辰时、巳时、午时、未时、申时、酉时；**黑夜**时辰：戌时、亥时、子时、丑时、寅时、卯时
5. 黑夜时辰玩家、召唤兽和怪物的普通物理攻击和法术驱动的物理攻击能力 **削弱20%**，有夜战和高级夜战技能不受此影响
