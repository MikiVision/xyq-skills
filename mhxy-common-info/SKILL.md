## 六、节日奖励

- **礼物领取地点**：长安（208,128）"节日礼物使者"NPC
- **领取条件**：等级≥30级
- **领取间隔**：每次领取间隔6小时
- **领取时间**：节日当天 00:00～24:00

### 礼物内容

1. 获得经验 = 等级×等级×60
2. 获得金钱 = 等级×等级×10
3. 获得一张宝图（请注意领取时物品栏需留出空位）

---

## 七、金钱携带上限

- **公式**：现金+存款 ≤ 等级×等级×2000+10000
- **最高携带上限**：3000万（其中银行存款最高1000万）
- 超出上限的金钱下线后不会消失，而是**自动转换成储备金**
- 金钱超出上限时会得到系统提示，建议下线前尽快处理

三界功绩是对玩家自强奋发的嘉奖，也是从事部分活动的前提。

### 获得途径

#### 1. 完成经验活动获得

| 活动名称 | 获得时机 | 获得三界功绩 |
|---|---|---|
| 师门 | 完成第10环 | 12点 |
| 渡劫师门 | 完成第10环 | 12点 |
| 官职任务 | 完成一轮单人任务 | 4点 |
| 官职任务 | 完成一轮多人任务 | 10点 |
| 社区任务 | 完成第10次 | 10点 |
| 副本 | 储备金模式完成1个 | 20点 |
| 奇谭 | 储备金模式完成1个 | 20点 |
| 种族任务 | 储备金模式完成第6~10环 | 1点/环 |
| 钟馗抓鬼 | 储备金模式完成第6~10环 | 1点/环 |
| 黑无常抓鬼王 | 储备金模式完成第6~10环 | 1点/环 |
| 二十八星宿 | 完成1个 | 2点/个 |
| 江湖传闻 | 完成1个 | 2点/个 |
| 战神任务 | 完成1个 | 1点/个 |
| 暗雷烧双 | ≥30级≥3人组队，有双倍或精修时间 | 9点(几率获得) |

#### 2. 寄售点卡获得

| 寄售成功 | 获得三界功绩 |
|---|---|
| 整张点卡 | (点数面额/5+2)点 |
| 月卡年卡 | (点数面额/5)点 |
| 散点 | (点数面额/5)点，取整 |

#### 3. 技能学习/修炼获得

| 途径 | 获得三界功绩 |
|---|---|
| 学习技能/帮派总管/紫衣盗/黑山老妖/乾元丹/五行修业 | (花费现金/3500+花费储备金/7000)点 |
| 点修炼/副本官员学习技能/召唤兽秘传 | (花费现金/5000+花费储备金/10000)点 |
| 坐骑技能/租赁仓库 | (花费现金/5000)点 |

#### 4. 消耗灵动仙玉

在霓裳姑娘和靓号使者处消耗灵动仙玉，可获得三界功绩：(仙玉总额/7)点，取整。每次获得上限为6000点。

### 消耗途径

#### 1. 现金模式任务消耗

| 任务名称 | 消耗时机 | 消耗三界功绩 |
|---|---|---|
| 种族任务 | 现金模式领取1环 | 1点/环 |
| 钟馗抓鬼 | 现金模式领取1环 | 1点/环 |
| 黑无常抓鬼王 | 现金模式领取1环 | 1点/环 |
| 副本 | 现金模式领取1个 | 20点 |
| 奇谭 | 现金模式领取1个 | 20点 |

#### 2. 金钱/物品任务消耗

| 任务名称 | 消耗时机 | 消耗三界功绩 |
|---|---|---|
| 跑镖（现金） | 领取1次 | 3点/次 |
| 打牌 | 领取1次 | 1点/次 |
| 钓鱼 | 购买鱼竿 | 40点/根 |
| 花香香 | 领取1次 | 1点/次 |
| 强盗宝图 | ≥50级时，每天第101次起领取1次 | 1点/次 |
| 强盗宝图 | ＜50级时，领取1次 | 1点/次 |

#### 3. 其他消耗

- 购买点卡：消耗(点卡面额×0.7)点，取整
- 藏宝阁上架金钱：暂扣(2×梦幻币数量/平均金价)点；售出则消耗，下架取回则返还

### 其他说明

领取奇遇任务、免资材点修炼、快速兑换修炼等方式不再消耗三界功绩。
