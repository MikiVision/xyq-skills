# 梦幻西游系统介绍

> 以下内容基于《梦幻西游》电脑版官网资料整理

## 一、聊天社交

### 1.1 聊天介绍

> 来源：https://xyq.163.com/introduce/sys/lt001.html

#### 聊天命令

频道选择：游戏中进行聊天时，可以针对不同场合以及聊天对象来选择发送的频道，可供选择的频道共有九个，进入游戏时系统默认的频道为"当前"，可以用鼠标左键点击画面左下方的频道按钮进行更换，或用快捷键 `Alt+<`、`Alt+>` 快速切换聊天频道。

| 频道 | 频道代码 | 频道信息 |
|------|----------|----------|
| 当前 | `//C` | 在当前频道发布的信息，可以被当前视野附近的所有玩家看到。 |
| 队伍 | `//T` | 组队时使用来和同组的队员交流，队伍以外的人无法看到聊天内容。 |
| 帮派 | `//B` | 用于同帮派的玩家相互交流，每个帮派都有自己独立的频道。 |
| 夫妻 | `//F` | 夫妻二人世界的专用频道，无论夫妻双方相隔多远，在这个频道发送的信息都能被对方看到。 |
| 私聊 | `//M` | 将某个特定玩家设为私聊对象后，就可以在这个频道单独发送信息给对方。 |
| 世界 | `//W` | 向整个游戏世界的玩家喊话，等级≥20的玩家才可以使用。每次发言需要消耗玩家的体力和活力各两点。 |
| 门派 | `//S` | 用于相同门派的玩家交流信息，各个门派的聊天频道相互独立。 |
| 传闻 | `//R` | 玩家可以在这个频道匿名发布消息，但有一定几率匿名失败而显示真实的人物名称，每次发言消耗2000两银子。 |

发言间隔：为了保证正常的聊天秩序，世界和传闻频道对连续发言规定了时间间隔，在其中任意一个频道发言后，另一个频道也要等待相应时间后才能发言。

#### 特殊指令

频道切换指令格式：`//频道代码 ××××××××`

文字格式：聊天时可以为消息内容设置文字格式，指令为 `#字母`，区分大小写。

| 格式 | 效果 |
|------|------|
| `#G` | 绿色(green) |
| `#B` | 蓝色(blue) |
| `#K` | 黑色(black) |
| `#Y` | 黄色(yellow) |
| `#W` | 白色(white) |
| `#b` | 闪烁(blink) |
| `#c` + 六位数字或A-F字母 | 自定义颜色，例如：`c008000`=暗绿色 |
| `#u` + 文字 + `#u` | 文字有下划线 |
| `#n` | 所有文字状态恢复正常 |
| `#r` | 文字换行 |
| `##` | 输出一个#号 |

#### 聊天查询

玩家可以查找到本次登陆以来某一个或多个指定频道的信息记录，还可将查到的信息以文本格式（.txt）输出保存。

#### 增强按钮

1. 右键点击人物后显示人物资料和增强按钮，右键锁定角色持续30秒。
2. 使用增强按钮可实现：给予、交易、申请、邀请、临时、好友、私聊、屏蔽。
3. "邀请"功能可邀请指定对象加入自己的队伍（邀请者必须已经组队）。
4. 点击私聊按钮后将立即将输入框切换到私聊频道，直接与该玩家私聊。

#### 聊天分栏

可通过聊天栏下方的"窗口分栏"按钮使用分栏显示功能。分栏状态下，聊天窗口中的"分界条"可以上下移动。系统公告将只显示在上方的界面中。聊天分栏只对聊天栏外置时有效。

#### 频道信息收集

聊天栏界面中有一个按钮 `[集]`，点击后弹出"频道信息收集"对话框。玩家可设置感兴趣的关键字，系统自动将带有这些关键字的信息筛选并保留下来。每次设置关键字需要消耗30点体力，关键字在下线后自动清空。

注意：频道信息收集功能只对打开的频道有效，不会收录传闻中的消息。完全相同的信息不会再次收录。信息在下线后不保留。

### 1.2 交识朋友

> 来源：https://xyq.163.com/introduce/sys/lt005.html

#### 添加好友

- 点击好友按钮（快捷键 `Alt+F`），选择"添加好友"，左键点击当前屏幕上的玩家即可添加。
- 也可以点选"查询好友"按钮，输入对方昵称或数字ID查询后添加。
- 在聊天栏中使用 `Shift+鼠标右键` 点击玩家名字，自动弹出添加好友界面。
- 角色等级≤50级，好友上限为50人；>50级时，每增加10级增加5个好友上限，好友人数上限为100人。

#### 友好程度

通过"友好度"体现，新添加好友的友好度为1。两人互为好友后，每共同战斗一场，双方友好度增加1点。互为好友的异性玩家可通过送红玫瑰提升友好度，每送一朵增加相互友好度99点。

#### 发送/接收信件

- 打开好友界面，在列表中点选好友名称，输入信件内容（最长230个汉字），点"发送"即可。
- 对方在线并且信件开关开启，可以马上收到信；不在线可在下次登陆时收到。
- 有信件送来时，好友按钮会闪烁。

#### 拒绝收信

- 开启"拒绝陌生人消息"开关（系统菜单 `Alt+S`），不再收到陌生玩家信件。
- 关闭"信件接收"开关，不再收到任何人的信件。

#### 临时好友

临时性的联系方式，临时好友名单中的联系人在玩家下线后消失。人数上限为100人，不可以转组，不会累加友好度。

#### 黑名单

系统将自动过滤黑名单内联系人发送的好友信息。人数上限为10人，不可以转组，需要手动删除。

## 二、交易商业

### 2.1 摆摊

> 来源：https://xyq.163.com/introduce/sys/sy001.html

#### 购买方法

- 购买物品：右键点击摊位招牌即可查看对方出售的物品，鼠标停留在物品上可以看到物品属性。
- 购买召唤兽：右键点击摊位招牌，再点上方的"召唤兽"按钮，可查看出售的召唤兽及其属性。

#### 摆摊打造

摆摊打造功能为打造师提供了便利的交易体验。

- 上架：使用鼠标左键点击界面右侧的摆摊制造类，在需要上架的项填入价格，选中勾选框，然后点击上架按钮。
- "保存设置"可保存当前价格信息，"读取设置"可读取已保存的价格信息（每次读取需花费20000两）。
- 使用摆摊打造服务需要缴纳一定的手续费。

#### 摆摊宝石合成

- 只有宝石工艺达到5级的玩家才能摆摊出售宝石合成服务。
- 宝石合成范围为1~10级，即最高可将10级宝石合成11级宝石。
- 红宝石、黄宝石、蓝宝石、绿宝石暂时不允许进行摆摊合成。

#### 摆摊收购

- 打开物品栏，点选"摆摊"→"收购摆摊"。
- 选择要收购的物品，输入购买单价和数量，点"发布"将物品摆上摊位。
- 每位玩家最多可收购10种物品，收购力总和不能超过30。
- 收购到物品时，摊主需支付物品价格7%的手续费（最低2000）。

### 2.2 商会

> 来源：https://xyq.163.com/introduce/sys/sy002.html

#### 开店介绍

玩家可以在游戏中开设和经营属于自己的商店，出售各类物品和召唤兽。商店分为物品类和唤兽类两个大类。

#### 申请开店

- 在长安城找到商会总管(326,18)申请登记店面。
- 申请人等级≥30，需支付开店所需资金。
- 每个游戏ID最多可同时开设经营两家商店。

#### 商店管理

- 店主可进行投入资金、取出资金、变更类型和宗旨等管理。
- 商店开设期间需支付维护费用和经营费用（经营费用仅在营业状态下计算）。
- 变更类型花费2万两，变更名称花费10万两。
- 每间商店最多容纳日常运营资金5000万两。

#### 基础营业费用与维护费用

- 每间普通柜台每天：50000两 × 商业指数
- 每间物品收购柜台每天：75000两 × 物品类商业指数
- 营业时每小时加收基础营业费用 × 商业指数

#### 出售与购买商店

- 店主不准备继续经营时，可将店铺出售（商店管理界面选择"申请出售"）。
- 商店售价不能低于商店内日常运营资金。
- 新开的商店在14天内不能出售。

#### 破产倒闭

商店在经营不善时进入破产保护期（14天）。保护期内支付 `15万 × 已进入保护期天数 × 当前商业指数 × 店面数` 的银两可退出保护期。过期则强制破产，店中物品和召唤兽无法取回。

#### 名店收藏

玩家可将经常光顾的店加入名店夹（最多5家）。店主可设立0~10%的名店优惠幅度。

#### 物品收购

- 开启物品收购功能需要有一间物品类商店。
- 每个物品收购柜台最多容纳20个收购物品。
- 增开一间物品收购柜台需缴纳 80万 × 物品类商业指数 的费用。
- 可收购物品：1级太阳石、1级月亮石、1级光芒石、1级黑宝石、1级红玛瑙、1级舍利子、1级符石、金柳露、超级金柳露、彩果、青龙石、白虎石、朱雀石、玄武石、定魂珠、金刚石、龙鳞、夜光珠、避水珠、符石卷轴。

#### 商会交易税

- 召唤兽店：若营业额≥500万，税率 = ((营业额-100万)÷20000万) × 100%（最高10%）
- 物品店：若营业额≥1500万，税率 = ((营业额-1000万)÷50000万) × 100%（最高4%）
- 采用退税制：固定收取每笔交易10%，每天晚上结算后将(固定税-交易税)返还。
- 收取的交易税总额50%返还给符合条件的店主平分，另50%抽奖形式返还给5名幸运店主。

#### 商会排名

排名规则：
1. 有号码的商店优先于没有号码的。
2. 有号码的商店按：营业店面数 > 每小时基础营业费 > 商店号码排序。
3. 没有号码的商店按：营业店面数 > 开店日期排序。
4. 店主可选择正序优先或反序优先。

号码竞拍：每天20:00-22:00开放一定数量号码竞拍。竞拍方式为限价竞拍或自动竞价。

### 2.3 钱庄

> 来源：https://xyq.163.com/introduce/sys/sy011.html

#### 钱庄介绍

钱庄是玩家存取虚拟货币的地方。各地钱庄的存款可以通用，存款没有利息。每个ID的存款上限为1000万两银子。每天取款额500万以下需执行部分解锁操作，500万以上需全面解锁。

| 钱庄名称 | 入口坐标 |
|----------|---------|
| 万字钱庄 | 建邺城（233，76） |
| 广源钱庄 | 长安城（512，194） |
| 长寿村钱庄 | 长寿村（147，91） |
| 傲来国钱庄 | 傲来国（107，56） |

#### 点卡金库

通过点卡寄售获得的游戏币存入点卡金库，只可领不可存。金库存放上限 = 人物等级² × 2000 + 10000（最高2000万）。

#### 钱庄特殊账户

等级≥60级的玩家可获得特殊账户。
- 存入上限：1亿 + 等级 × 500万
- 每日取出上限：2000万 + 等级 × 100万
- 每个服务器每天可从特殊账户中累计取出-存入的总现金为10亿

#### 银券买卖

开通特殊账户后可购买银券。银券分1个月、2个月和3个月三种类型，到期可兑换成现金进入特殊账户。每月发行时确定利率，月初买利息多，月尾买利息少。

### 2.4 股票系统

> 来源：https://xyq.163.com/introduce/sys/sy015.html

《梦幻西游》股票系统参考现实社会股票交易市场制作，股票价值完全由市场买卖需求决定。

#### 交易账户

- 在长安城(507,195)找股票交易员创建，等级≥60级。
- 各等级账户最大现金额：60~79级2亿、80~99级4亿、100~119级6亿、120~139级8亿、140~155级10亿。
- 每日可取金额：从60级起每天限取1000万，每提升10级增加1000万，最高每日1亿。

#### 股票交易

- 交易委托：限价委托方式，价格在上一日收盘价±10%范围内。
- 撮合成交：按"价格优先、时间优先、交易量优先"原则，当"买入价≥卖出价"时成交。
- T+1机制：当天买进的股票，第二天可卖出。
- 交易税：每笔成功交易缴纳总价0.5%（买卖均收取）。
- 手续费：单笔委托成交后固定收取1000两。

#### 交易时间

| 时间 | 安排 |
|------|------|
| 11:40-12:00 | 接受交易委托，于12点产生开盘价 |
| 12:00-14:00 | 开市 |
| 14:00-19:00 | 休市 |
| 19:00-21:00 | 开市 |
| 21:00-11:40 | 休市、清除本日剩余的交易委托 |

#### 红利分配

每支股票代表一个商家，每月公布月季报告，将净利润归入资本公积金库。红利分配事件每月随机触发，由十大股东决议是否分配。分配日收盘后统一派发，并进行"除息"处理。

### 2.5 仙玉系统

> 来源：https://xyq.163.com/introduce/sys/sy016.html

#### 系统介绍

仙玉是上古遗留之物，具有吸取天地灵气、凝聚三界浩然之力的能力。角色等级≥20级可获得。

#### 仙玉类型

| 类型 | 获得方式 | 应用范围 |
|------|---------|---------|
| 灵动仙玉 | 账号充值（专用/通用点数）兑换 | 购买或赠送锦衣、购买祥瑞、彩饰、炫卡等 |
| 静息仙玉 | 充值年卡/月卡、办理绿色通道 | 购买锦衣、祥瑞（将军令用户绑定≥90天、等级≥50）、彩饰等。不能用于赠送，不能购买西域焰火 |
| 免费仙玉 | 三界功绩累积每达300点生成1个（兑换时需消耗20点三界功绩） | 购买锦衣、祥瑞、彩饰等。不能用于赠送及转服烟花竞拍 |

灵动仙玉与静息仙玉之间不会互相转化。

#### 兑换仙玉

在宝象国(48,39)的霓裳姑娘或宝象国(57,25)和仙缘洞天(61,126)的桃源仙子处兑换。仙玉与游戏点数兑换比例为1:1。每个ID每天最多兑换200000个。

#### 附加效果

当角色仙玉总数量（灵动+静息+免费）≥1000时，获得当前人物最高属性+1点属性点的附加效果。低于1000时失去此效果。附加效果不会累加。

## 三、帮派系统

### 3.1 帮派建立

> 来源：https://xyq.163.com/introduce/sys/bp001.html

#### 建立帮派

- 申请人等级≥40，当前没有加入帮派。
- 在长安城找"帮派主管"(384,270)申请，缴纳150万梦幻币。
- 输入帮派名称（最多七个汉字）和宗旨。
- 需要10名等级≥20的响应者，48小时内响应人数不足则建立失败，费用不退。
- 新服开放第5天中午12点方允许建立帮派。

#### 帮派的作用

- 每个帮派有自己的集会场所和帮派频道。
- 帮派成员可学习辅助技能（打造技巧、烹饪技巧、炼金术、健身术等）。
- 可通过完成帮派任务获得经验值奖励。
- 帮派提供福利：领取薪水、修理装备、购买药品、恢复状态等。
