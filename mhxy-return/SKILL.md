# 梦幻西游老玩家回流知识库

## 一、老玩家回流

### 获取条件

- 久未登录游戏且等级≥60的角色
- 每年获取回流服务次数有限制

### 回流服务期限

角色回归首次登录时，梦幻精灵将与角色签订回流服务，具体时长以游戏内回流主界面显示为准。

### 使用方式

1. 获得回流服务后首次登录，梦幻精灵寄出"老玩家回流请柬"
2. 右键使用请柬打开回流主界面
3. 游戏界面左上角也显示回流主界面入口

### 回流奖励

- 完成"每日活跃"回流任务获得活跃度
- 达到一定活跃度可领取经验或储备金奖励
- 每日活跃度达到100可抽取回流大转盘
- 可点击任务右上角替换按钮自选任务

### 回流特权

#### 专属服务

| 特权名称 | 说明 |
|---|---|
| 专属酒肆 | 专属酒店休息功能 |
| 专属巫医 | 专属超级巫医功能 |
| 泡泡助战 | 师门任务战斗中超级泡泡助战 |
| 临时符 | 四张专属临时符 |
| 精灵护佑 | 每日首次正常阵亡无损失 |
| 专属藏宝阁 | 定制推荐的藏宝阁页面 |

#### 加成特权（部分需满活跃度3次解锁）

| 特权名称 | 说明 |
|---|---|
| 经验加成 | 完成部分任务经验+20%（同类效果不叠加） |
| 组队加成 | 好友组队经验+15%，带队+20% |
| 帮派加成 | 组队参加帮战，队员额外10%经验+20%帮贡 |
| 房屋特权 | 免费将房屋耐久和清洁度恢复到50 |

### 注意事项

1. 每年回流服务次数有限
2. 转服或藏宝阁交易后会取消回流服务，所有进度清空
3. 回流奖励的回梦丹存放在宝石商人处，注意存储上限

## 二、特殊服务器

### 活动专区——武神坛

武神坛服务器是服务器联赛专用服务器，仅在联赛期间向参赛玩家和记者开放。其他服务器玩家可在各自服务器内找NPC观看比赛与蚩尤封印。

### 活动专区——玲珑岛

临时活动专用服务器，不定期举办各种临时线上活动，除活动专用账号外普通账号无法登入。

### 活动专区——寻梦园

临时活动专用服务器，不定期举办各种临时线上活动，除活动专用账号外普通账号无法登入。

### 测试区——梦幻测试

- 免费服务器，用于稳定性和玩法测试
- 不定期开放，不定期清档
- 不保证运行稳定性与数据完整性

### 测试区——开发测试

- 免费服务器，针对性测试新系统
- 仅对申请通过的玩家开放
- 测试期间开放时间：10:00-24:00
- 报名：长安城（312,56）开发测试使者
- 目前仅开放将军令用户报名，名额有限先到先得
- 提交意见：长安城多个情报专线NPC处（420,210 / 230,165 / 180,95 / 315,55 / 450,95 / 400,60 / 445,105）
- 测试数据不影响原服务器，测试结束后清档
