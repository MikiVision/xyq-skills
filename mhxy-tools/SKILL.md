# 梦幻西游辅助工具知识库

## 一、梦幻精灵

梦幻精灵是游戏内的自动问答助手，可回答常见问题。

### 使用方法

1. 打开好友列表（快捷键 `Alt+F`），点击好友栏第一页第一位的"梦幻精灵"；或使用 `Alt+V` 快捷键直接联系
2. 在消息界面输入问题，点"发送"
3. 稍后即可收到回复

### 注意事项

- 用简洁语言描述问题，避免错别字
- 仅回复常见问题，突发情况/BUG请联系在线GM或客服
- 如有无法回答的常见问题，可通过客服专区提交

## 二、团队契约

防止组队跳车，通过交纳保证金约束队伍成员。

### 申请条件

- 队伍人数≥4人
- 携带足够的保证金

### 如何申请

1. 组好队伍后到长安找**团队合约担保人（244,171）**申请
2. 队长选择契约类型后，全部队员同意才生效

### 契约规则

- 合约期分别为 **1、2、4小时**，保证金可选 **2万、5万、10万** 两
- 每个玩家一次只允许有一个团队契约
- 违约被投诉3票后，没收保证金并扣除20人气
- 合约期满后可领回保证金

## 三、保险加锁

为重要虚拟物品（道具、召唤兽）加锁保护。

### 加锁流程

1. 打开物品栏，选择"加锁"按钮
2. 选择准备加锁的物品或召唤兽，点"加锁"
3. 输入并确认加锁密码（请勿使用通行证账号或密码）

### 解锁

- 点选"解锁"，输入密码验证成功后即可选择物品解锁
- 重新登陆游戏后解锁需再次输入密码

### 修改密码

- 在解锁界面点"修改密码"，验证旧密码后设置新密码，每次消耗20点体力

### 强行解除

- 遗忘密码可申请"强行解除"，消耗100点体力
- 申请15天后再次点击"强行解锁"，系统清除旧密码

## 五、小号绑定

保护仓库小号安全，NPC位于**建邺城（139,25）"游戏角色绑定使者"**。

### 绑定规则

- 一个小号只能绑定一个大号，一个大号可绑定多个小号
- 绑定关系永远无法解除
- 绑定后，小号只能与绑定大号及同一大号的其他小号互相给予和交易

### 绑定方法

- 大号（60级以上）组队带领未绑定小号（10级以下），找绑定使者申请
- 需双方确认

### 绑定后限制

1. 无法升级
2. 无法申请自杀
3. 无法丢弃/购买/使用物品
4. 只能与绑定关系内角色交易
5. 无法打造、缝纫、典当、烹饪、炼药、炼妖、合成宝石
6. 无法领取新手任务

## 七、角色交易

通过藏宝阁完成角色买卖，ID从卖家通行证转移到买家通行证。

### 购买流程

1. 登陆藏宝阁购买
2. 用购买角色进入游戏，找**贸易车队总管**（长安433,182 / 长安271,45 / 建邺衙门8,12）取出角色

### 购买注意

- 同账号同服最多6个角色，超出需自杀腾位
- 勿用需同时上线的角色购买（如师徒、夫妻等）
- 不限制等级，10级以下可在建邺衙门取出

### 出售条件

1. 等级≥69级
2. 已使用将军令或密保卡≥75天
3. 未处于体验状态
4. 金钱未超上限，无系统负债
5. 未报名移民
6. PK开关关闭，未被通缉

### 出售流程

1. **登记角色**（14天登记期）
2. **寄售到藏宝阁**（登记完成后7天内）
3. **上架设价**
4. **收款或取回**

### 手续费

- 交易额3000元以下：60元
- 交易额3000元以上：交易额×1%+30元
- 买家不收费

## 十一、时间锁

使用时间锁的物品或召唤兽，在设定时间内（1到180天）不能解除保护。

### 如何使用

1. 找**房都尉（长安375,73）**
2. 全面解锁后，选择"我要加时间锁"，输入锁定时间（1~180天）
3. 选择要加时间锁的道具或召唤兽

### 如何解除

1. 设定时间到期后，找房都尉
2. 全面解锁后，选择"我要解时间锁"（解除后自动使用物品锁）

### 限制

- 时间到期前不能解除
- 加锁物品不能给予、交易、丢弃、出售、合成、寄售或改名
- 装备、灵饰和召唤兽装备加持时间锁需要锻造等级≥1

## 十二、高价值流转频率控制

### 高价值召唤兽规则

符合以下条件之一即为高价值召唤兽：
1. 神兽
2. 泡泡灵仙
3. 技能数≥7
4. 灵性值≥80
5. 特定高级技能总数≥4，且参战等级≥45级
6. 新服（≤90天），参战等级及召唤兽等级都≥45级的宝宝

**限制**：交易流通后48小时内不能再次流通（除非返回原角色）。藏宝阁买入72小时内不能流通。

### 高价值物品规则

**武器和防具**满足之一：
1. 锻炼宝石等级≥6
2. ≥80级且特技/特效/套装总数≥2，或携带无级别、愤怒、罗汉金钟、晶清诀、琴音三叠、破血狂攻
3. ≥100级且锻炼宝石等级≥4

**灵饰**满足之一：
1. ≥100级且3条附加属性
2. 精炼等级≥5
3. 特效"超级简易"

**召唤兽装备**：镶嵌等级≥5

### 交易金钱保管

高价值交易成功后，卖家所得梦幻币暂存于**长安钱庄（512,194）保险箱**。
- 使用将军令/密保卡验证后可提前领取
- 交易次日24:00后自动发放
