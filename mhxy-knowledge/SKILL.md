# 梦幻西游装备与宝石知识库

本知识库涵盖梦幻西游端游中装备系统和宝石系统的核心机制与实战建议。

---

## 一、装备基础体系

### 1.1 装备等级与分类

| 等级段 | 装备等级 | 适用等级范围 |
|--------|---------|-------------|
| 低级 | 0-40级 | 新手至中期过渡 |
| 中级 | 50-80级 | 中期主力装备 |
| 高级 | 90-110级 | 高级玩家核心 |
| 顶级 | 120-160级 | 顶级玩家追求 |

装备分类：
- **武器**：各门派专属武器，决定物理/法术伤害输出
- **衣服/铠甲**：提供防御和气血
- **头盔**：提供防御属性和额外加成
- **腰带**：主要加气血和防御
- **鞋子**：提供速度和防御
- **项链**：提供法术伤害/物理伤害加成
- **配饰（戒指/手镯）**：提供多种属性加成

### 1.2 装备属性类型

基础属性：
- **伤害/命中/法术伤害**：攻击类属性
- **防御/躲避/法术防御**：防御类属性
- **气血/魔法/速度**：基础三围属性

附加属性：
- **体质/魔力/力量/耐力/敏捷**：五维加点属性
- **特效**：附加概率触发效果
- **特技**：主动释放的技能效果

### 1.3 装备品质

- **白装（普通）**：无附加属性
- **绿装（优良）**：1条附加属性
- **蓝装（精良）**：2条附加属性
- **金装（极品）**：多条高级附加属性

## 二、装备打造系统

### 2.1 打造流程

1. **准备材料**：百炼精铁/玄铁矿（武器），锦缎/蚕丝（衣服），需要对应等级材料
2. **选择配方**：到铁匠铺/裁缝铺选择对应装备
3. **打造执行**：消耗活力 + 材料 + 金钱
4. **装备鉴定**：使用鉴定符或到NPC处鉴定

### 2.2 打造技巧

- **活力值**：打造消耗活力，活力上限与等级相关
- **技能等级**：打造技能等级影响出极品概率
- **环装打造**：做师门/任务获得的环装有机会出极品
- **炼妖石**：提升打造品质的辅助道具

### 2.3 装备鉴定

鉴定后装备会随机获得附加属性：
- 属性条数随机（1-6条不等）
- 属性类型随机
- 属性数值在一定范围内随机
- 有几率出现特效/特技

### 3.3 不磨装备

- **机制**：不磨装备使用不消耗耐久度
- **来源**：天台/藏宝阁/副本产出
- **价值**：不磨属性大幅提升装备保值性
- **等级**：不磨分为 1-10 级不磨，数字越大越稀有

### 3.4 简易装备

- **机制**：附带"简易"特殊词条，降低装备使用等级要求
- **用途**：低等级玩家可以使用高等级装备，适合小号
- **举例**：150级简易14，实际使用等级要求降低14级

### 3.5 无级别装备

- **机制**：无装备等级限制，任意等级可穿戴
- **价值**：梦幻西游中最珍贵的装备类型之一
- **特点**：
  - 极其稀有，出现概率极低
  - 多出现在高等级装备上
  - 价值极高，顶级无级别可达数十万甚至百万人民币
- **鉴别**：装备信息中显示"无等级限制"

## 四、宝石系统

> 以下内容基于《梦幻西游》电脑版官网资料整理（来源：xyq.163.com/introduce/dj008.html）

### 4.1 基本概念

- 宝石是用来**加强装备**的特殊材料，游戏中共有 **11种** 宝石，每种附加属性各不相同。
- 宝石具有**等级**属性，装备具有**锻炼等级**属性。每镶嵌一颗宝石，装备的锻炼等级 +1。
- **同种类宝石，无论等级多少，镶嵌效果都相同。** 但只有 `宝石等级 =（装备锻炼等级 + 1）` 时，才可以进行镶嵌。
- 一件装备最初只能镶嵌1颗1级宝石，之后可以陆续镶嵌2级、3级宝石，**不允许跃级镶嵌**。

### 4.2 宝石种类与效果（共11种）

| 宝石名称 | 可镶嵌装备 | 镶嵌效果（每颗） |
|---------|-----------|----------------|
| 光芒石 | 铠甲、腰带 | 增加 40 点气血上限 |
| 太阳石 | 武器、头盔 | 提高 8 点伤害力 |
| 月亮石 | 铠甲、头盔 | 提高 12 点防御力 |
| 红宝石 | 饰品、鞋子 | 增加 4% 的火系法术吸收率 |
| 黄宝石 | 饰品、鞋子 | 增加 4% 的雷系法术吸收率 |
| 蓝宝石 | 饰品、鞋子 | 增加 4% 的水系法术吸收率 |
| 绿宝石 | 饰品、鞋子 | 增加 4% 的土系法术吸收率 |
| 黑宝石 | 腰带、鞋子 | 提高 8 点速度 |
| 红玛瑙 | 武器、头盔 | 提高 25 点命中 |
| 舍利子 | 铠甲、饰品 | 提高 6 点灵力 |
| 神秘石 | 武器、腰带、鞋子 | 提高 20 点躲避力 |
| 翡翠石 | 铠甲 | 增加 12 点法术防御力 |

**关于法术吸收率的说明**：
表中的火系、水系等法术，指**怪物和召唤兽**所施展的法术（如烈火、水漫金山等），不包括三昧真火、龙卷雨击等门派法术。如果成功吸收对方的法术，可恢复自身一定的气血。

### 4.3 宝石获取方式

1. **长安城宝石商人**：出售部分一级宝石，种类和数量每 10 分钟变化一次。价格根据出售情况变化——同类宝石每出售一颗价格上涨，每次刷新时有所降低。
2. **任务产出**：完成某些任务时有可能获得较高级的宝石，例如挑战28星宿、封妖等。

### 4.4 宝石镶嵌

#### 镶嵌方法
在**铁匠铺**里将宝石镶嵌到装备上，操作方法和打造基本相同，需要把打造的原料换成装备和宝石。每种宝石只能镶嵌在**固定类型**的装备上（如光芒石只能镶嵌铠甲和腰带）。

#### 镶嵌规则

1. 每件装备**最多可镶嵌 2 种类型**的宝石。例如铠甲已经镶嵌了光芒石和月亮石，就不能再镶嵌舍利子。
2. 每件装备都具有"锻炼等级"属性，初始为 0 级。每次镶嵌宝石后，锻炼等级 +1。
3. **宝石等级 > 装备锻炼等级** 时，才可以镶嵌。
4. 当装备的 **锻炼等级 ≥ 装备要求等级 / 10** 时，无法再镶嵌宝石。无级别限制的装备无此限制，但镶嵌时也要求宝石等级 > 装备锻炼等级。

### 4.5 宝石合成

在**铁匠铺**里可将两颗宝石合成为一颗高一级的宝石。

**合成条件**：
1. 两颗原料宝石的**种类相同**
2. 两颗原料宝石的**等级相同**

**合成结果**：
- 合成后宝石等级 +1，种类与原料宝石相同
- 合成过程中**有一定几率失败**，失败会丢失两颗宝石中的一颗
- 合成宝石等级越高，成功率越低
- 学习剧情技能"**宝石合成**"可提高成功率

### 4.6 宝石拆分（碎石锤）

玩家可以使用"**碎石锤**"拆除镶嵌在装备/武器上的宝石（武器仅允许对只镶嵌了一种宝石的使用），降低装备锻炼等级，同时清除宝石效果。

#### 碎石锤属性
每个碎石锤有两项属性：**等级**与**类型**。

#### 使用条件
1. 碎石锤等级 ≥ 装备宝石锻炼等级
2. 碎石锤所属类型与装备镶嵌宝石类型相同

#### 使用效果
- 每个碎石锤使用后可拆除 1 级宝石
- 使用后碎石锤消失，同时降低装备 1 级锻炼等级及所拆宝石对应的属性
- **普通碎石锤拆除后宝石不返还**

#### 碎石锤获取方式
1. **百晓积分**兑换
2. 拥有 **5级宝石工艺** 的玩家，去傲来国蝴蝶妹妹处花费 150 点活力制造
3. 两个**相同等级、相同类型**的碎石锤可在打铁炉合成为更高一级的碎石锤，消耗体力 = 碎石锤等级 × 10，合成不需要任何技能，**不会失败**

#### 其他说明
1. 只有镶嵌一种宝石的武器才可以使用碎石锤拆除
2. 拆除成功后，锻炼等级 -1，同时扣除宝石所加成的效果，**宝石不予以返还**

### 4.7 精制碎石锤

精制碎石锤是碎石锤的升级版，**可以返还宝石**。

#### 获取方式
找到**傲来国（60，54）的蝴蝶妹妹**，花费一定的金钱和体力，将"碎石锤"升级为同等级的"精制碎石锤"。

#### 功能说明
1. 使用精制碎石锤可将装备上的宝石效果**全部清除**，同时**返还一颗**与锻炼宝石类型相同、等级为锻炼等级的宝石（有两种宝石效果则随机返还其中一种类型的宝石）
2. 使用条件：精制碎石锤等级 ≥ 装备宝石锻炼等级，且不能对有两种宝石效果的武器使用
3. 成功使用后，精制碎石锤会消失
4. 精制碎石锤**无法通过两两合成**的方式获得，只能通过碎石锤升级获得
5. 精制碎石锤**无法对失败三次的装备使用**

## 五、装备评估与选择

### 5.1 装备价值评估维度

1. **基础属性数值**：越接近上限越好
2. **附加属性条数**：条数越多越好
3. **附加属性质量**：属性数值是否接近满值
4. **特效/特技**：有高价值特效特技大幅增值
5. **不磨等级**：不磨装备溢价明显
6. **宝石孔数量**：2孔优于1孔
7. **外观**：部分特殊外观有收藏价值

### 5.2 各门派武器选择建议

| 门派类型 | 武器关注属性 | 核心宝石 |
|---------|------------|---------|
| 物理门派（大唐/狮驼岭） | 伤害/命中/速度 | 太阳石 + 黑宝石 |
| 法术门派（魔王/龙宫） | 法术伤害/速度 | 神秘石 + 舍利子 |
| 封系门派（盘丝/地府） | 速度/法术伤害 | 神秘石 |
| 辅助门派（普陀/化生） | 速度/法术伤害 | 神秘石 |

### 5.3 性价比建议

- **新手期**：不必追求极品，够用即可，把资源投入宝石
- **中期过渡**：选择性价比高的蓝装，搭配5-6级宝石
- **高级玩家**：追求特效/特技金装，宝石尽量满级
- **土豪玩家**：无级别/顶级不磨 + 满级宝石

## 六、常见问题解答模板

### Q：XX门派应该用什么装备？
回答框架：
1. 先确认门派定位（物理/法术/封系/辅助）
2. 推荐武器属性优先级
3. 推荐防具属性优先级
4. 推荐宝石搭配方案
5. 给出不同预算的方案

### Q：这件装备值多少钱？
回答框架：
1. 分析基础属性数值（与满值对比）
2. 统计附加属性条数和质量
3. 是否有特效/特技及其价值
4. 是否不磨及不磨等级
5. 宝石孔数量
6. 参考服务器物价给出估价范围

### Q：宝石应该怎么打？
回答框架：
1. 确认角色门派和定位
2. 确认当前装备情况
3. 按优先级推荐宝石种类
4. 建议宝石等级目标
5. 给出性价比最优的升级路径
