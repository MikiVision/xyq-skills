# 梦幻西游召唤兽相关

> 以下内容基于《梦幻西游》电脑版官网资料整理

## 一、宠物与召唤兽概述

宠物是玩家在游戏世界里的忠实伙伴，它们的形象来自中国传统的十二生肖，分为鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪，每个宠物的形象都很可爱逗趣。

### 如何获得宠物

玩家最初进入游戏会随机获得一只宠物，之后找到建邺城的**宠物仙子（59，117）**，可以找她免费更换一次宠物（之后更换宠物需要交纳5万两银子的手续费）。共有12种宠物供玩家选择，宠物领养后不能遗弃。

### 宠物的相关数值

- **名字**：宠物领养时的默认名字和生肖名相同，玩家可以随时给自己的宠物起喜欢的名字。在"宠物"界面（快捷键 `Alt+P`）的"名字"一栏输入宠物的名字，再点"更改名字"即可。
- **等级**：根据宠物的经验计算，经验达到要求后自动升级，等级会影响宠物可以携带召唤兽的数量和耐力上限。
- **耐力**：存在当前耐力和耐力上限。当前耐力是指当前可用的耐力值，耐力上限是当前耐力可以达到的最大值。宠物驯养召唤兽的时候需要消耗当前耐力，可以通过喂养宠物口粮来恢复。随着宠物等级的提升，耐力上限会增加，每升一级耐力值上限加1。
- **经验**：根据玩家在线时间长短计算，玩家每在线一分钟，宠物经验增加一点，经验达到要求后自动升级。

### 宠物的初始属性

| 属性 | 初始值 |
|------|--------|
| 名字 | 宠物生肖名称 |
| 等级 | 0 |
| 耐力 | 10 |
| 经验 | 0 |

### 宠物的作用

**携带召唤兽**：玩家在战斗中捕捉到的召唤兽是通过宠物携带的。随着宠物等级的提升，可携带召唤兽的数量也会增加。等级 ≤19 的宠物最多可携带3只召唤兽，20级时可以携带数量为4只，以后每增加20级携带的召唤兽数量加1。

**驯养召唤兽**：召唤兽可以协助玩家战斗，但是在忠诚度比较低的时候可能会不愿参战，通过宠物驯养可以增加召唤兽的忠诚度。打开召唤兽界面（快捷键 `Alt+O`），在列表中选择要驯养的召唤兽名字，再点下方的"驯养"即可，每驯养一次可以恢复1点的忠诚度。驯养召唤兽会损耗宠物的耐力，损耗程度受到召唤兽等级的影响。

**炼妖合成**：两个召唤兽可以通过炼妖而合成为一个新的召唤兽，但是只有等级 ≥30 的召唤兽才能被用来合成。合成后的召唤兽资质、能力是根据合成前的召唤兽各资质、能力获得的，有可能比以前的资质更高。合成后原召唤兽技能有一定几率会遗传给新的召唤兽，还有一定几率合成出0级的召唤兽宝宝。

**其他操作**：
- 观看宠物：打开宠物界面，点击"观看"按钮可以将宠物放到地上观看。
- 收回宠物：打开宠物界面，点击"隐藏"按钮可以将观看的宠物收回。
- 喂养宠物：可以通过喂养宠物口粮来增加宠物的耐力，宠物口粮在杂货店有售。

## 二、出现场景

游戏中某些野外场景分布着不同种类的怪物，根据场景的不同，出现怪物的能力也有一定差别。主要怪物类型包括：

- **普通怪物**：攻击和防御能力受到等级影响，具有法术技能的怪物可以施展法术进行攻击。
- **怪物头领**：综合战斗能力要强于其它同类怪物，大部分会施展法术。
- **怪物宝宝**：宝宝类怪物被捕捉后成为0级的召唤兽。在处于异常状态时不能被捕捉。
- **怪物精灵**：精灵在战斗的第二回合会自爆，以牺牲自己来攻击对方全体。不能被捕捉。
- **变异宝宝**：由寻常怪物变异而成，与同类怪物形体相同但颜色独特，比较稀有，捕捉后成为0级宝宝。
- **泡泡宝宝**：泡泡是比较特殊的召唤兽，不会出现在固定场景里。玩家在普通战斗中有一定几率遇到，捕捉成功后为0级宝宝。泡泡没有携带等级的要求，不可用于炼妖合成。

玩家可以使用**摄妖香**避免遇敌，使用后可以不再遇到场景等级低于自己等级的怪物暗雷。

### 暗雷头领

在暗雷场景中，会遇到一些特殊的NPC：

1. **狡猾的貔貅**：进入战斗后隐身，如果在两个回合内降伏它，队伍里会随机有一个队员得到藏宝图。三个回合后貔貅显形。
2. **告密的貔貅**：杀掉所有小怪后逃掉，在原地会出现明雷boss，成功击杀会得到丰厚奖励。
3. **友好的貔貅**：指名挑战玩家，被指名的玩家和它们比试后会遁走。
4. **懦弱的貔貅**：会影响伙伴士气，需要在逃跑前消灭它们。
5. **黑鱼仔貔貅**：成群结队，会不断召唤兄弟姐妹战斗。

## 三、携带等级

召唤兽能否携带受玩家的等级限制，等级越高，能携带的召唤兽种类也越多。

- 玩家不可以捕捉高于自己等级10级的召唤兽。
- 不可以捕捉自己等级不够支持的类型的召唤兽。
- 召唤兽超过自己的等级5级以上，则战斗中不再增加经验值。

## 四、变异召唤兽

在每一种召唤兽中，隐藏着一些鲜为人知的特殊品种。它们由于受到各种不明因素的长期影响而产生了变异，在其成长性、资质、寿命及外观上，都会表现出与众不同的特质。

### 特点

- 变异召唤兽的颜色是独有的，从外观颜色上即可辨别。
- 变异后的召唤兽有更大可能获得较高的资质和寿命，因此是召唤兽中的珍稀品种。
- 玩家在普通战斗中有非常小的几率遇到变异召唤兽，成功捕捉后即可作为自己的召唤兽。
- 刚捕捉的变异召唤兽为0级宝宝，携带要求和同类普通召唤兽一样。
- 可以学习魔兽要诀，但**不能用来炼妖合成**，也不能当作任务需求的召唤兽使用（除召唤兽修炼任务外）。

### 净瓶玉露

- **净瓶玉露**：可以将参战等级小于95级的变异召唤兽重置为0级变异宝宝。
- **超级净瓶玉露**：可以将参战等级 ≥95 级的变异召唤兽重置为0级变异宝宝。
- 使用时有一定几率获得新的变异召唤兽颜色。

## 五、召唤兽宝宝

召唤兽宝宝由于是0级的召唤兽，成长以后有更多可以自由分配的属性点（每升一级有5个属性点），因此是玩家战斗中的好帮手。

- **怪物宝宝**：宝宝类怪物被捕捉后成为0级的召唤兽。在处于异常状态时不能被捕捉。
- **变异宝宝**：由寻常怪物变异而成，与同类怪物形体相同但颜色独特，比较稀有，捕捉后成为0级宝宝。
- **泡泡宝宝**：泡泡是比较特殊的召唤兽，不会出现在固定的场景里。玩家在普通战斗中有一定几率遇到泡泡宝宝，捕捉成功后为0级宝宝。用两只任意类型的怪物宝宝合成时，也有一定几率得到泡泡宝宝。

## 六、召唤兽资质、技能、成长

### 资质属性

- **攻击资质**：召唤兽攻击相关属性资质的体现，资质越高攻击越强，给敌人的伤害也就越大。
- **防御资质**：召唤兽防御相关属性资质的体现，资质越高防御越强，受到的物理类伤害也就越小。
- **体力资质**：召唤兽气血相关属性资质的体现，体力资质越高，HP越多。
- **法力资质**：召唤兽魔法值相关属性资质的体现，法力资质越高，MP越多，法术威力也越强。
- **速度资质**：召唤兽速度相关属性资质的体现，速度资质越高每点敏捷影响的速度越大。
- **躲避资质**：召唤兽躲避力相关属性资质的体现，该资质越高越容易躲开敌人的物理攻击。

### 召唤兽技能

召唤兽的技能只有在战斗中才会发挥作用，主要起到直接伤害、恢复自身属性和增强防御等作用。在召唤兽界面中点选"查看资质"可以打开资质列表，显示当前召唤兽的各项资质和技能图标。

## 七、炼妖打书

### 炼妖合成

两个召唤兽可以通过炼妖而合成为一个新的召唤兽，但是只有等级 ≥30 的召唤兽才能被用来合成。合成后的召唤兽资质、能力是根据合成前的召唤兽各资质、能力获得的，有可能比以前的资质更高，更适合战斗。原召唤兽技能有一定几率会遗传给新的召唤兽，还有一定几率合成出0级的召唤兽宝宝。

两个召唤兽炼妖合成时，增加**天眼珠、三眼天珠、九眼天珠**任意一种，有一定几率合成出**阴阳伞**召唤兽。

### 打书（魔兽要诀）

- **魔兽要诀**能赋予召唤兽某种技能，但每次使用同时会消除该召唤兽的一项原有技能。技能数量 ≤2 的召唤兽有一定几率不会消除原有技能。
- 魔兽要诀包含大多数宝宝技能，有高级和低级之分。低级要诀一般通过挖宝获得，高级要诀通过挖5宝获得。
- **金柳露**：可以把95级以下的普通召唤兽变为0级。
- **超级金柳露**：用于合成 ≥95 级的召唤兽。

### 炼妖操作方法

打开宠物界面，选择"炼妖"，在弹出的炼妖界面中会显示出召唤兽物品列表。如果是想把两个召唤兽合成，在列表中先后点选两个召唤兽，再点击"炼妖"按钮；如果想用物品和召唤兽合成，可以分别点选召唤兽和合成物品，再点"炼妖"即可。

## 九、召唤兽修炼

### 任务简介

- **地点**：天宫（227，71）
- **任务人**：马真人
- **领取要求**：玩家等级 ≥65 级，每次消耗50点体力和10万两银子。

### 修炼类型

共有四种修炼类型可以选择：

- **物理攻击**：提高召唤兽的物理攻击能力，每修炼一级，攻击时HP伤害结果提高 2%+5点。
- **物理防御**：提高召唤兽抵抗物理攻击的能力，每修炼一级，受到物理攻击时HP伤害结果降低 2%+5点。
- **法术攻击**：提高召唤兽的伤害能力和封类法术的成功率，每修炼一级，法术的HP伤害结果增加 2%+5点。
- **法术防御**：提高召唤兽抵抗法术攻击的能力，提高抗封几率，每修炼一级，受到的法术伤害结果降低 2%+5点；被封几率降低2%。

每项修炼最高可以达到**20级**。当玩家等级 ≤ 当前修炼类型等级×5+20时，完成任务不再增加该项修炼的经验。

> **特别说明**：玩家在这些项目上的修炼不会影响到召唤兽本身属性，当召唤兽移交给他人时，战斗能力将按新主人的修炼等级重新计算。

### 任务种类

- **找人**：领取任务时会被告知需要寻找的NPC名称，找到后对话即可。
- **寻物**：找到任务要求的物品，给予相关NPC即可。

### 任务奖励

每完成一环任务可获得修炼经验，当前环数越高增加的经验也越多。完成第100环任务时，如果积分达到要求，有一定几率得到高级指南书或精铁，以及修炼果、修炼经验或3级家具设计图。修炼果服用后增加某种修炼经验150点。

### 秘传修炼

- **任务时间**：每天10:00到24:00
- **参与条件**：等级 ≥65 级，单人任务
- **奖励**：每次完成后获得最少150点，最多200点召唤兽修炼经验
- 每人每天最多领取6次秘传修炼任务

## 十、召唤兽饰品

召唤兽饰品是一种用于加强召唤兽效果表现，丰富现有召唤兽装扮的物品。

### 饰品获得

1. 通过门派师傅用门贡进行礼物兑换有几率获得"玄天残卷·上卷""玄天残卷·中卷""玄天残卷·下卷"。
2. 凑齐上中下三本残卷可以与建邺城（63，115）的**宠物仙子**兑换1颗"**圣兽丹**"。
3. 将圣兽丹与特定召唤兽炼妖，即可合成出该召唤兽的饰品。
4. 与圣兽丹炼妖合成的召唤兽会被消耗。

### 饰品属性

- 每只召唤兽只能佩戴一个召唤兽饰品。
- 召唤兽佩戴饰品后会立刻显示装扮效果，在战斗及非战斗中均可显示。
- 召唤兽饰品有指定使用的召唤兽类型，可以给该类型的普通或变异召唤兽佩戴。
- 饰品拥有**玄天灵力**属性，战斗中召唤兽被攻击累积一定次数后，灵力会流失。灵力为0时饰品失去作用。

### 玄天灵力修补

- 通过铁匠铺打铁炉进行修补操作（有一定失败几率，失败会减少灵力上限）。
- 也可以使用**珍珠**做为修补材料，修补成功率为100%。

### 饰品染色

- 找建邺城（63，115）的宠物仙子进行染色。
- 每次染色消耗3个彩果。
- 召唤兽饰品有3种颜色可供选择（不包含默认颜色）。

## 十二、召唤兽进阶与特性

### 所需条件

1. 携带等级 ≥45 级的召唤兽。
2. 召唤兽内丹已达上限并且低级内丹达到5层。

当满足条件的召唤兽在明暗雷战斗进行历练时，就能够感受到《异兽录》残卷的气息，进入进阶之路。玩家可以携带该召唤兽前往**仙缘洞天（147，86）**寻找**御兽仙**。

### 灵性

- 所有感应到《异兽录》气息的召唤兽都会增加一个名为"灵性"的属性，**每点灵性可以使召唤兽额外获得2点潜能点**。
- 当灵性小于50，可以使用**易经丹**在96小时后为召唤兽增加10点灵性，御兽仙可以加快吸收速度。
- 当灵性达到**50**以后，召唤兽获得一个焕然一新的造型作为奖励。如果想继续使用进阶前的造型，可以前往宠物仙子处切换。
- 当灵性大于50时，可以使用**清灵仙露**继续增加灵性。
- 当灵性提升到**80**，召唤兽可以领悟独一无二的"**特性**"。

### 特性封印及解封

- 当召唤兽习得特性后，若正向效果未达最大值，可以在御兽仙处**封印**该特性。
- 若正向效果已达最大值，该特性将无法被封印。
- 召唤兽特性被封印后将无法生效，并获得随机牢固值。
- 给被封印的召唤兽喂食**易经丹**可以减少封印牢固值。当牢固值降至0，封印解除，被封印特性的正向效果变为最大值，并**随机出现一个负面效果**。

## 十三、召唤兽栏扩充

### 扩充方式

打开召唤兽栏，点击界面上的"开启新召唤兽栏"即可进行扩展。每扩展成功一个新的召唤兽栏位，即可多携带一个召唤兽。

### 扩展规则

1. 召唤兽栏最大数目为**12栏**（包含初始8栏），即最多可扩展4栏。
2. 每次扩展增加1栏，即可多携带一只新的召唤兽。
3. 扩展后的栏位为**永久使用**。
4. 扩展到4个栏位后不再显示扩展按钮。
5. 扩展栏位后只增加携带数量，召唤兽参战数量不变。
6. 摆摊时携带的全部召唤兽都将显示，但最多上架8只。

### 扩展价格

| 扩展次数 | 价格 |
|----------|------|
| 第1次 | 1000万 |
| 第2次 | 3000万 |
| 第3次 | 6000万 |
| 第4次 | 10000万 |

> 优先扣除储备金。
